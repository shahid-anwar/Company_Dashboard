# 🏢 Company Directory Dashboard

A modern, responsive React-based company directory with advanced filtering and search capabilities.

## 🚀 Features

- ✅ **Responsive Design** - Works seamlessly on mobile, tablet, and desktop
- ✅ **Advanced Filtering** - Filter by industry, location, and search by name
- ✅ **Debounced Search** - Optimized search with 500ms debounce
- ✅ **Dynamic Sorting** - Sort by company name, employees, or founding year
- ✅ **Context API State Management** - Clean, centralized state management
- ✅ **Modern UI** - Gradient backgrounds, glassmorphism effects
- ✅ **Grid/List View Toggle** - Switch between card and list layouts
- ✅ **Loading States** - Smooth loading animations
- ✅ **Error Handling** - Graceful error messages

## 🛠️ Tech Stack

- **Frontend**: React.js (Vite)
- **Styling**: Tailwind CSS
- **State Management**: React Context API
- **Data Source**: Static JSON file
- **Deployment**: Vercel

## 📦 Installation

```bash
# Clone the repository
git clone https://github.com/Roopesh127/company-dashboard.git

# Navigate to project directory
cd company-dashboard

# Install dependencies
npm install

# Run development server
npm run dev
```
